# Transcriptie: C6 — Claude Settings

## Wat leer je in deze module?
Je leert welke instellingen er beschikbaar zijn in Claude.ai, wat ze doen en welke je moet kennen voor dagelijks gebruik. Na deze module heb je je account correct ingesteld voor jouw situatie.

## Waar vind je de instellingen?

Klik op je profielfoto of initialen rechtsboven in Claude.ai. Kies "Settings". Je ziet een aantal categorieën: Profile, Memory, Privacy, Appearance en Billing.

## Profile

Hier stel je je naam en e-mailadres in. Je kunt ook een korte beschrijving van jezelf toevoegen — wie je bent, wat je doet — zodat Claude die context heeft in elk gesprek. Dit is handig als je wilt dat Claude altijd weet dat je bijvoorbeeld een makelaar bent of een juridisch medewerker. Je hoeft die context dan niet elke keer opnieuw te geven.

## Memory

Geheugen staat standaard uit. Als je het inschakelt, onthoudt Claude relevante informatie uit je gesprekken voor toekomstige gesprekken. Claude kiest zelf wat hij onthoudt — jij kunt ook handmatig informatie toevoegen of verwijderen.

Praktisch: schakel geheugen in als je Claude regelmatig gebruikt en je niet telkens dezelfde context wilt herhalen. Maar controleer af en toe wat er in het geheugen staat — Claude kan dingen onthouden die je liever niet terugziet.

Je kunt het geheugen ook volledig wissen via de instellingen. Handig als je een schone start wilt.

## Privacy

Hier vind je de instelling voor het gebruik van je gesprekken voor modeltraining. Bij een gratis account worden gesprekken standaard gebruikt. Je kunt dit uitschakelen. Bij zakelijke abonnementen (Teams, Enterprise) is dit standaard al uitgeschakeld.

Dit is een instelling die je direct moet controleren als je Claude gebruikt voor professioneel werk.

## Appearance

Kies tussen licht en donker thema. Geen functioneel verschil — puur persoonlijke voorkeur.

## Billing

Hier zie je je huidige abonnement, je factuurgeschiedenis en kun je upgraden of opzeggen. Als je vragen hebt over je abonnement of factuur, staat hier ook de link naar support.

## Taal en interface

Claude.ai is beschikbaar in meerdere talen. De interface is Engels maar Claude zelf reageert in de taal die jij gebruikt. Als jij Nederlands schrijft, antwoordt Claude in het Nederlands. Je hoeft daar niets voor in te stellen.

## Keyboard shortcuts

Claude heeft een paar handige sneltoetsen die je workflow versnellen:
- **Enter** — verstuur bericht
- **Shift + Enter** — nieuwe regel zonder te versturen
- **Ctrl/Cmd + K** — nieuw gesprek starten
- **Ctrl/Cmd + /** — snelkoppelingen bekijken

## Samenvatting

De belangrijkste instellingen om te kennen: schakel de privacy-instelling voor modeltraining uit als je professioneel gebruik maakt van Claude, voeg je profiel toe voor automatische context, en overweeg geheugen in te schakelen als je Claude regelmatig gebruikt. De rest van de instellingen zijn optioneel en puur voor persoonlijke voorkeur.
