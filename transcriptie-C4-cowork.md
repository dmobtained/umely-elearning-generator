# Transcriptie: C4 — Cowork: Taken delegeren aan Claude

## Wat leer je in deze module?
Je leert wat Cowork is, hoe je taken delegeert aan Claude via de desktop-app, en welke soorten taken zich goed lenen voor Cowork. Na deze module kun je Claude inzetten voor repetitieve taken op je computer die je normaal handmatig doet.

## Wat is Cowork?

Cowork is de functie in de Claude desktop-app waarbij Claude niet alleen antwoorden geeft maar ook daadwerkelijk stappen uitvoert op je computer. Hij kan bestanden lezen, mappen aanmaken, documenten bewerken en bestanden opslaan. Je geeft een opdracht, Claude maakt een plan, jij keurt goed, Claude voert uit.

Het verschil met gewone Chat: bij Chat geeft Claude je een antwoord dat jij dan zelf uitvoert. Bij Cowork voert Claude het zelf uit — met jouw toestemming.

## Hoe werkt het in de praktijk?

Je opent de desktop-app en kiest voor Cowork. Je geeft Claude toegang tot een map op je computer. Daarna geef je een opdracht in gewone taal.

Voorbeeld voor een makelaarskantoor: "Lees alle PDF-bestanden in de map Bezichtigingen van deze maand en maak voor elk bestand een korte samenvatting als tekstbestand in de map Samenvattingen."

Claude laat je eerst zien wat hij van plan is. Je bevestigt, en dan voert hij de stappen één voor één uit. Je ziet wat hij doet.

## Welke taken zijn geschikt?

Cowork werkt goed voor:
- Het verwerken van grote aantallen bestanden (samenvatten, hernoemen, sorteren)
- Het aanmaken van mappenstructuren
- Het omzetten van bestandsformaten (tekst naar gestructureerd overzicht)
- Het doorzoeken van mappen op specifieke informatie

Cowork werkt minder goed voor:
- Taken waarbij je specifieke software nodig hebt (Excel-formules berekenen, presentaties opmaken)
- Taken waarbij internettoegang nodig is
- Alles wat real-time data vereist

## Altijd in controle

Claude vraagt altijd om bevestiging voordat hij iets doet dat niet ongedaan gemaakt kan worden — zoals een bestand overschrijven of verwijderen. Je kunt op elk moment stoppen. Je bent altijd de baas.

Maak wel altijd een back-up van belangrijke bestanden voordat je Claude op een grote map loslaat. Niet omdat hij fouten maakt, maar omdat automatisering altijd risico's heeft en voorzichtigheid een goede gewoonte is.

## Beperking: geen toegang tot systemen buiten je computer

Cowork werkt alleen met bestanden op je eigen computer of in mappen die je hebt gedeeld. Claude heeft geen toegang tot je e-mail, je CRM, je boekhoudpakket of andere systemen — tenzij je die expliciet koppelt via MCP (zie module E1). Voor die integraties is meer setup nodig.

## Samenvatting

Cowork laat Claude taken uitvoeren op je computer in plaats van alleen antwoorden geven. Je delegeert een taak, Claude maakt een plan, jij keurt goed, Claude voert uit. Geschikt voor bestandsverwerking, mapbeheer en repetitieve documenttaken. Claude vraagt altijd om bevestiging bij ingrijpende acties. Maak altijd een back-up voordat je met Cowork aan de slag gaat op een grote map.
