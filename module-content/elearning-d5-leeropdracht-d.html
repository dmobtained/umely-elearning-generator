Je bepaalt of Claude Code past bij jouw use case, installeert het via VS Code, schrijft een CLAUDE.md en gebruikt Plan Mode. Aan het einde ken je het pad van idee naar werkende setup.

Weet je het al?

Deze leeropdracht is het meest waardevol voor wie regelmatig met bestanden werkt, scripts schrijft of taken wil automatiseren. Werkt jouw use case puur met tekst in de browser? Dan is C8 wellicht voldoende, maar het kan geen kwaad dit te verkennen.

Na deze leeropdracht heb je:

  • Bepaald welk deel van jouw use case geschikt is voor Claude Code
  • Claude Code geïnstalleerd via de VS Code-extensie
  • Een CLAUDE.md geschreven die aansluit op jouw use case
  • Plan Mode gebruikt om een taak te plannen voordat Claude hem uitvoert

Claude Code heeft meerwaarde als je use case een van deze drie dingen bevat: je werkt met meerdere bestanden tegelijk, je schrijft of past scripts aan, of je voert herhalende taken uit die je nu handmatig doet. Kijk naar je use case uit A4 en bepaal of een of meer van deze patronen daarin zitten.

Wanneer wel, wanneer niet

Bestanden verwerken
Je hebt een map met rapporten die je wekelijks hernoemt, doorzoekt of samenvoegt. Claude Code kan dit automatiseren zodat je het niet meer handmatig doet.

Scripts schrijven
Je wil een script dat elke maandag automatisch een overzicht maakt van nieuwe bestanden in een map. Claude Code schrijft en test dat script in je eigen omgeving.
Klik voor uitleg
Taken automatiseren
Je doet dezelfde reeks stappen elke week opnieuw: exporteren, hernoemen, samenvatten, versturen. Claude Code kan die reeks als een workflow bouwen.

Installeer Claude Code via VS Code

Jouw use case:

De VS Code-extensie is de eenvoudigste manier om met Claude Code te starten. Geen terminal, geen Node.js-installatie. Je installeert VS Code, zoekt de extensie op en logt in met je Anthropic-account.

Installatiestappen

1

Download VS Code

Ga naar code.visualstudio.com en download de versie voor jouw systeem. Installeer en open het programma.

2

Open het Extensions-paneel

Klik op het vier-blokjes-icoon in de linkerzijbalk, of gebruik Ctrl+Shift+X (Windows/Linux) of Cmd+Shift+X (Mac).

3

Zoek en installeer Claude Code

Typ 'Claude Code' in de zoekbalk. Klik op de extensie van Anthropic en klik op Install.

4

Log in met je Anthropic-account

Na installatie verschijnt een inlogprompt. Gebruik het e-mailadres en wachtwoord van je Anthropic-account op claude.ai.

Lukt de installatie niet?

Zie module D2 voor de volledige installatie-uitleg inclusief de terminal-route als alternatief. De extensie-route in D2 is identiek aan wat hier beschreven staat.

Minimaal 10 tekens.

Schrijf een CLAUDE.md voor jouw use case

✓

Jouw use case:

Claude Code begint elke sessie zonder geheugen van vorige gesprekken. CLAUDE.md is de oplossing: een bestand in de hoofdmap van je project dat vertelt wie je bent, wat het project is en hoe Claude moet werken. Je schrijft het eenmalig. Claude gebruikt het automatisch bij elke start.

Maak een nieuwe map aan op je computer voor je project. Dat is de map waar je ook je bestanden zult opslaan.

Kopieer deze template en vul hem in voor jouw use case
# [Naam van jouw use case] ## Context [Wie ben jij, wat doe je, voor wie werk je] ## Wat Claude moet doen [De taak die Claude uitvoert in dit project] ## Formaat [Hoe de output eruit moet zien] ## Wat Claude moet weten [Specifieke regels, toon, aandachtspunten]
1

Kopieer de template

Gebruik de knop hierboven om de template naar je klembord te kopieren.

2

Maak een nieuwe map aan

Maak een map aan op je computer voor dit project, bijvoorbeeld op je bureaublad of in Documenten.

3

Maak CLAUDE.md aan in VS Code

Open de map in VS Code via File > Open Folder. Maak een nieuw bestand aan: Ctrl+N (of Cmd+N op Mac), sla op als CLAUDE.md.

4

Vul de template in

Vervang alle plaatshouders door inhoud die past bij jouw use case. Hoe concreter, hoe nuttiger het is voor Claude.

Moet beginnen met # en minimaal 20 tekens bevatten.

Gebruik Plan Mode voor je use case

✓
✓
✓
4
Jouw use case:

Plan Mode laat Claude eerst een aanpak voorstellen voordat hij iets uitvoert. Je geeft de opdracht, Claude schrijft een stap-voor-stap plan en wacht op jouw goedkeuring. Ideaal voor complexe taken of taken waarbij bestanden worden aangepast: je ziet wat er gaat gebeuren voordat het gebeurt.

Typ deze opdracht in Claude Code: pas de inhoud aan voor jouw use case
Maak eerst een plan voor: [beschrijving van je use case-taak]
1

Open Claude Code in VS Code

Klik op het Claude-icoon in de linkerzijbalk. Je ziet het chatvenster van Claude Code.

2

Activeer Plan Mode

Druk op Shift+Tab om Plan Mode in te schakelen, of begin je opdracht met "Maak eerst een plan voor:". Claude schrijft een plan zonder iets uit te voeren. De exacte keybinding kan per versie variëren: controleer via /help in Claude Code.

3

Lees het plan

Claude geeft een stap-voor-stap aanpak. Lees welke stappen hij van plan is te zetten en of dat klopt met wat je wil.

4

Stuur bij of keur goed

Vertel Claude wat je anders wil, of typ "ga verder" als het plan correct is. Claude voert pas iets uit na jouw akkoord.

Wanneer gebruik je Plan Mode?

Gebruik Plan Mode bij elke nieuwe taak die meer dan één stap heeft, zeker als bestanden worden aangepast, verplaatst of aangemaakt. Bij leestaken (samenvatten, overzicht maken) is het optioneel maar nog steeds nuttig.
