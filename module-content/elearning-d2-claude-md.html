Module 1 van 4

Wat is CLAUDE.md?

CLAUDE.md is een tekstbestand dat je aanmaakt in een projectmap. Elke keer dat je Claude Code opstart in die map, leest hij dit bestand automatisch in. Alles wat erin staat, instructies, context, regels, voorkeuren, weet Claude Code dan direct.

Vergelijk het met een interne briefing die je aan een nieuwe medewerker geeft: dit zijn onze regels, dit is hoe wij werken, dit zijn de dingen die je altijd moet doen of nooit moet doen.

Module 3 van 4

CLAUDE.md aanmaken

Twee manieren

A

Via Claude Code zelf

Open Claude Code in je projectmap en typ: "Maak een CLAUDE.md aan met de volgende inhoud: [jouw instructies]"

B

Handmatig

Open een tekstverwerker, schrijf je instructies en sla het op als CLAUDE.md (hoofdletters, zonder spaties) in de hoofdmap van je project.

Tip

Houd CLAUDE.md onder de 200 regels. Schrijf alleen wat echt relevant is. Alles wat je nooit gaat gebruiken is ruis die Claude Code kan afleiden.

Onderhoud en best practices

Behandel CLAUDE.md als een levend document. Pas het aan als je merkt dat Claude Code een fout maakt die je niet wilt herhalen. Bijvoorbeeld: "Voeg toe aan CLAUDE.md: maak nooit een bestand aan met de naam 'definitief'. Gebruik altijd een datum in de bestandsnaam."
