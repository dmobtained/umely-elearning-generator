Leeropdracht A: kies je use case en bouw je eerste prompttemplate

Dit is de eerste leeropdracht van de cursus. Je kiest hier een echte werktaak als use case. Die taak bouw je door alle volgende leeropdrachten heen uit.

Vul je naam in om verder te gaan.
Of kies een voorbeeld:
Beschrijf je use case om verder te gaan.
Wat is een goede use case?

Je use case is een terugkerende werktaak die je wekelijks of dagelijks doet. Kies iets concreets, geen grote vage ambitie, maar een specifieke klus. Die use case bouw je door de hele cursus heen uit.

Hoe deze leeropdracht werkt

1
Open Claude.ai in een nieuw tabblad en zet het naast dit scherm.
2
Elke fase geeft je een kopieerbare prompt die je invult en verstuurt.
3
Kom terug, vul je verificatieveld in en ga door naar de volgende fase.
Geschatte tijd: 20-30 minuten

Na deze leeropdracht heb je:

  • Een concrete use case gekozen die je door de hele cursus heen gebruikt
  • Gezien wat er gebeurt als Claude geen context krijgt
  • Je prompt verbeterd met context, opdracht en formaat
  • Een herbruikbare prompttemplate gebouwd voor je eigen werktaak

Dit is je nulmeting. Je stuurt bewust een vage, onduidelijke prompt voor je use case. Geen uitleg over wie je bent, wat je precies wilt of hoe het resultaat eruit moet zien.

Claude zal iets terugsturen, maar het antwoord zal waarschijnlijk generiek zijn. Dat is het punt: je ziet straks precies wat context doet.

Tip

Weersta de neiging om meer uitleg te geven. Een bewust vage prompt is de bedoeling. Hoe vaag je het houdt, hoe duidelijker het contrast wordt in fase 2.

Minimaal 20 tekens. Dit bewijst dat je de prompt daadwerkelijk hebt gestuurd.

Leermoment: Het antwoord was waarschijnlijk generiek of niet precies bruikbaar. Claude wist niet wie je bent, wat het doel was of hoe het resultaat er uit moest zien. In fase 2 los je dat op.

Jouw use case:

Een goede prompt hoef je maar één keer goed te schrijven. In deze fase zet je de verbeterde prompt uit fase 2 om naar een template met {{variabelen}}. Die variabelen zijn de plekken die per situatie veranderen.

Volgende keer hoef je de structuur niet te herschrijven: je vult alleen de variabelen in.

Kopieer deze prompt naar Claude.ai
Maak van mijn verbeterde prompt een herbruikbare template. Zet de onderdelen die per situatie veranderen tussen dubbele accolades als {{variabele}}. De vaste structuur en toon blijven staan. Voorbeeld van het format: Jij bent {{rol}} voor {{doelgroep}}. Schrijf {{formaat}} over {{onderwerp}}. Toon: {{toon}}. Geef de volledige template terug, zodat ik hem direct kan kopiëren en bewaren.
1
Kopieer de prompt hierboven.
2
Ga naar Claude.ai: het gesprek met je verbeterde prompt staat er nog.
3
Stuur op en lees de template die Claude teruggeeft.
4
Test hem direct: gebruik de template opnieuw met andere invulling voor dezelfde use case.
5
Kopieer de volledige template en plak hem hieronder.
Sla je template op

Bewaar de template in een document, notitie of Claude-project. Je hebt hem nodig in de volgende leeropdrachten. Wie de template kwijtraakt, moet het werk opnieuw doen.

Minimaal 40 tekens.
