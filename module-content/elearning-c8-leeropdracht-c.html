Leeropdracht C: Claude volledig instellen voor jouw use case

Je hebt in A4 een use case gekozen en in B4 een veilige werkopdracht gemaakt. Nu configureer je Claude volledig voor die use case: juist gereedschap, profiel, memory en een eigen Project met Custom Instructions.

Pak je voorbereiding erbij

1
Pak je notities uit A4 (use case) en B4 (werkopdracht) erbij. Die gebruik je in elke fase.
2
Open Claude.ai in een nieuw tabblad en houd het naast dit scherm.
3
Elke fase heeft concrete stappen. Voer ze uit, kom terug en ga door.
Vul je naam in om verder te gaan.
Vul je use case in om verder te gaan.
Geschatte tijd: 35-50 minuten

Na deze leeropdracht heb je:

  • Het juiste Claude-gereedschap gekozen op basis van jouw use case
  • Je use case daadwerkelijk uitgevoerd in dat gereedschap
  • Je profiel ingesteld en een relevante memory opgeslagen
  • Een Project aangemaakt met Custom Instructions specifiek voor jouw use case

Vraag Claude zelf wat het beste gereedschap is voor jouw use case
Ik wil Claude gebruiken voor de volgende use case: [plak hier je use case] De drie Claude-opties die ik heb zijn: 1. claude.ai (web-app) — altijd beschikbaar, geen installatie, Projects mogelijk 2. Desktop-app — snel toegankelijk, Cowork beschikbaar, Projects mogelijk 3. Claude in Excel/PowerPoint — rechtstreeks in Office, geen Projects Welk gereedschap past het beste bij deze use case? Leg in drie korte punten uit waarom.
1
Kopieer de prompt, vervang de placeholder door jouw use case.
2
Stuur hem naar Claude.ai en lees het advies.
3
Maak je eigen keuze: je hoeft Claude niet te volgen als jouw situatie anders is.
Tip

Als je use case zich afspeelt in Excel of PowerPoint en een directe Office-integratie beschikbaar is in jouw regio en abonnement, kun je die kiezen. Anders is de web-app het sterkste startpunt: altijd beschikbaar, geen installatie nodig en je kunt er ook Projects aanmaken.

Minimaal 20 tekens. Beschrijf je keuze en de reden.

Je profiel vertelt Claude wie je bent, wat je doet en hoe je graag werkt. Dit geldt voor alle gesprekken. Memory laat Claude relevante details onthouden tussen gesprekken in. Beide instellingen staan in je claude.ai-instellingen.

3
Vul de voorzet hieronder aan met jouw gegevens en plak hem in het profielveld.
4
Sla op.
Voorzet voor je profiel: pas aan op jouw situatie
Ik ben [naam], [functie] bij [organisatie of type bedrijf]. Ik gebruik Claude voornamelijk voor: [noem 2-3 use cases, inclusief die uit A4]. Ik werk graag met [toon: formeel / bondig / uitgebreid — kies wat past]. Schrijf antwoorden standaard in het Nederlands tenzij ik iets anders vraag.

Controleer daarna of Claude je profiel heeft opgeslagen met deze prompt:

Wat weet je over mij op basis van mijn profiel?

Een Project geeft Claude vaste context voor jouw use case. De Custom Instructions die je schrijft gelden voor elk gesprek in dat Project. Je hoeft je situatie, toon en werkwijze niet meer te herhalen.

Project aanmaken

1
Ga naar claude.ai en klik op 'Projects' in de linkerzijbalk.
2
Klik op 'New Project' en geef het een naam die je use case beschrijft.
3
Klik op 'Add instructions' en vul de template hieronder in.
4
Sla op, open een nieuw gesprek in het Project en test het met je werkopdracht uit B4.
Template voor Custom Instructions: vul in voor jouw use case
Je assisteert [naam], [functie] bij [organisatie of type bedrijf]. Doel van dit Project: [beschrijf de use case in één zin — wat ga je hier altijd mee doen?] Schrijfstijl: [formeel / informeel / zakelijk-vriendelijk], altijd in het Nederlands. Context die je altijd moet meenemen: - [bijv. doelgroep van de output] - [bijv. vast format of structuur die je gebruikt] - [bijv. terminologie of afkortingen die gelden in deze context] Reageer altijd direct op de vraag. Geen overbodige inleiding.

Test het Project met deze verificatieprompt, vanuit het nieuwe Project:

Beschrijf in één zin wat je over mij weet en hoe je mij in dit Project assisteert.
Tip

Maak voor elke werkcontext een apart Project. Een Project voor klantenrapportages, een voor interne communicatie, een voor offertes. Elk met eigen Custom Instructions die op die context zijn afgestemd.

Minimaal 20 tekens. Dit bewijst dat de Custom Instructions actief zijn.
