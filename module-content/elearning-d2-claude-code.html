Module 1 van 5

Wat is Claude Code?

Claude Code is een command-line tool waarmee je Claude via de terminal gebruikt. De terminal is een tekstgebaseerd scherm op je computer. In tegenstelling tot de web-app kan Claude Code meerdere stappen achter elkaar uitvoeren: bestanden lezen, mappen aanmaken, tekst schrijven, bestanden aanpassen.

Claude Code is in de eerste plaats gebouwd voor developers. Maar een deel van de functies, zoals bestanden hernoemen, mappen organiseren en herhalende taken automatiseren, is ook bruikbaar voor niet-technische gebruikers die gewone zinnen kunnen typen. Je hoeft geen code te kennen om daarmee aan de slag te gaan.

Module 2 van 5

De terminal

Een terminal is een programma op je computer waar je tekst-opdrachten intypt. Op Windows heet het "Command Prompt" of "PowerShell". Op Mac heet het "Terminal". Je kunt het openen via de zoekfunctie van je computer.

Het ziet er kaal uit. Geen knoppen, geen menu's. Maar dat is precies wat het krachtig maakt. Je typt een opdracht in gewone taal, Claude voert het uit.

Terminal openen

W

Windows

Zoek op "PowerShell" of "Command Prompt" in de zoekbalk en open het programma.

M

Mac

Zoek op "Terminal" via Spotlight (Cmd + Spatie) en open het programma.

Tip

De terminal ziet er misschien intimiderend uit, maar je hoeft geen technische commando's te kennen. Claude Code begrijpt gewone taal.

Module 3 van 5

Installatie

Claude Code installeren

1

Installeer Node.js

Download de LTS-versie (stabiel) via nodejs.org. Installeer het zoals elk ander programma.

2

Open de terminal en installeer Claude Code

Typ: npm install -g @anthropic-ai/claude-code en druk op Enter. Wacht een paar minuten.

3

Start Claude Code

Typ: claude en druk op Enter. Log in met je Anthropic-account. Klaar.

Let op
