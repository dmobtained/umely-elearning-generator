Module 1 van 4

Plan Mode

Standaard voert Claude Code direct uit wat je vraagt. Plan Mode verandert dat: Claude maakt eerst een gedetailleerd plan en wacht op jouw goedkeuring voordat hij begint.

Je schakelt Plan Mode in door op Shift+Tab te drukken in Claude Code. Je kunt ook woorden als "plan" of "think about" in je prompt gebruiken om Claude richting planning te sturen. In Plan Mode analyseert Claude de taak, maakt een plan en vraagt om goedkeuring voordat hij begint met uitvoeren.

Het Plan Mode-proces

Shift+Tab

Je activeert Plan Mode

►

Plan

Claude beschrijft elke stap

►

Review

Jij keurt goed of past aan

►

Uitvoering

Claude voert het plan uit

Tip

Gebruik Plan Mode voor taken met veel stappen, taken op grote aantallen bestanden, of elke keer dat je niet zeker weet wat Claude gaat doen. Het kost iets meer tijd maar geeft je volledige controle.

Module 3 van 4

Hooks: automatische acties

Hooks zijn shell-commando's die jouw systeem automatisch uitvoert op specifieke momenten in de Claude Code levenscyclus. Je configureert ze in settings.json. Het zijn geen acties die Claude zelf uitvoert, maar scripts die jouw systeem draait op events zoals PreToolUse, PostToolUse en Notification.

Klik op een type hook voor een voorbeeld

PreToolUse
Draait voordat Claude een tool gebruikt. Bijvoorbeeld: een linting-script dat automatisch controleert voordat een bestand wordt bewerkt.
Klik voor voorbeeld
PostToolUse
Draait nadat Claude een tool heeft gebruikt. Bijvoorbeeld: automatisch tests uitvoeren nadat een bestand is gewijzigd.
Klik voor voorbeeld
Notification
Draait wanneer Claude Code een melding stuurt. Bijvoorbeeld: een geluidssignaal afspelen of een Slack-bericht sturen wanneer een lange taak klaar is.
Klik voor voorbeeld
Let op

Hooks zijn shell-commando's die op je systeem draaien. Test ze goed voordat je ze inschakelt. Een fout in een hook kan je workflow blokkeren. Begin eenvoudig en bouw uit naarmate je meer ervaring krijgt.

Module 4 van 4

Combineren voor maximale controle

Je kunt commando's en Plan Mode combineren. Schakel Plan Mode in met Shift+Tab en geef vervolgens je opdracht. Bijvoorbeeld: "Verwerk de bestanden in de map Inbox en voer daarna de tests uit." Claude maakt dan een plan met alle stappen, zodat je precies weet wat er gaat gebeuren voordat hij begint.

Samenvatting: wanneer gebruik je wat?

FunctieWanneer gebruiken
Plan ModeBij complexe taken of als je niet zeker bent wat Claude gaat doen
/clearAls het gesprek onoverzichtelijk is geworden
/costOm je API-kosten in de gaten te houden
HooksVoor automatische shell-commando's op lifecycle-events (configureer in settings.json)
Tip

Begin met Plan Mode bij je eerste grote taken. Configureer hooks in settings.json als je merkt dat je steeds dezelfde controles handmatig uitvoert. Gebruik /cost om bewust te blijven van je API-gebruik.
