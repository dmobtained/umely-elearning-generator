Onderdeel 1 van 4

Wat is de desktop-app?

De Claude desktop-app is een zelfstandig programma dat je installeert op je Mac of Windows-computer. Hij werkt onafhankelijk van je browser en staat altijd klaar via de taakbalk of menubalk.

Je downloadt hem via claude.ai/download. Installatie duurt minder dan een minuut. Je logt in met hetzelfde account als de web-app. Alle gesprekken en Projects zijn op beide beschikbaar.

Tip

De desktop-app en de web-app zijn geen aparte producten. Ze delen hetzelfde account en dezelfde gesprekken. Het verschil zit in de extra mogelijkheden die de desktop-app biedt via directe toegang tot je computer.

Drie functies in de desktop-app

1

Chat

Hetzelfde gespreksvenster als de web-app. Typen, vragen, bestanden uploaden. Maar sneller beschikbaar en kan bestanden op je computer lezen zonder uploaden.

2

Cowork

Claude voert taken daadwerkelijk uit op je computer: bestanden lezen, mappen aanmaken, documenten bewerken. Altijd met jouw bevestiging.

3

Code

Voor het schrijven, uitleggen en debuggen van code. Relevant als je interne tools of automatiseringen wilt bouwen.

Chat en de voordelen

De Chat-functie in de desktop-app werkt hetzelfde als de web-app. Je typt een vraag, Claude antwoordt. Bestanden uploaden kan via het paperclip-icoon. Gesprekken worden opgeslagen.

Het verschil zit in de toegankelijkheid en in een optionele extra: als je de desktop-app toestemming hebt gegeven om een map te lezen, kan Claude bestanden in die map gebruiken zonder dat je ze opnieuw hoeft te uploaden.

Voordelen van Chat via de desktop-app

1

Sneller starten

Geen browser openen, geen tabblad zoeken. Klik op het icoon in de taakbalk en je bent direct bij Claude.

2

Bestanden op je computer

Als je de app toestemming hebt gegeven voor een map, kan Claude die bestanden direct lezen. Handig als je regelmatig met dezelfde documenten werkt.

3

Minder afleiding

De app heeft geen andere tabbladen, meldingen of browserextensies. Je werkt in een rustige omgeving.

Onderdeel 3 van 4

Cowork: taken uitvoeren op je computer

Cowork is de functie waarbij Claude niet alleen antwoorden geeft, maar daadwerkelijk stappen uitvoert op jouw computer. Hij kan bestanden lezen, mappen aanmaken, documenten bewerken en bestanden opslaan.

Het verschil met Chat is cruciaal: bij Chat geef je jij het antwoord uit. Bij Cowork voert Claude het zelf uit, maar altijd met jouw toestemming per stap.

Chat vs Cowork: het verschil

Chat

Claude geeft antwoord. Jij leest het en voert zelf de actie uit.

►

Cowork

Claude maakt een plan. Jij keurt het goed. Claude voert het uit op je computer.

Let op

Claude vraagt altijd om bevestiging voordat hij ingrijpende acties uitvoert, zoals een bestand overschrijven of verwijderen. Lees altijd het plan door voordat je bevestigt. Maak een back-up van belangrijke bestanden voordat je Cowork op een grote map loslaat.

3

Selecteer de map waar Claude mee mag werken. Dit doe je eenmalig per map.

4

Geef je opdracht

Beschrijf in gewone taal wat je wilt. Claude toont het plan. Jij bevestigt.

Code en wanneer welk product?

De Code-functie is voor mensen die code schrijven of willen leren schrijven. Claude kan helpen bij het genereren, uitleggen, debuggen en verbeteren van code. Dit is relevant als je interne tools of automatiseringen wilt bouwen, ook voor kleinere projecten.

Niet-developers hoeven hier niets mee te doen. Maar het is goed te weten dat het bestaat. Als je op een punt komt dat je wilt automatiseren, is dit het startpunt.

Tip

Je hoeft geen programmeur te zijn om de Code-functie nuttig te vinden. Claude kan ook uitleggen wat een stukje bestaande code doet, of helpen bij het aanpassen van een script dat je van een collega hebt gekregen.

Wat kan Code?

Code genereren
Beschrijf wat je wilt en Claude schrijft de code. Van een eenvoudig Python-script tot een complete functie in JavaScript. Handig voor interne tools of automatiseringen.
Klik voor uitleg
Code uitleggen
Plak een stukje code en vraag Claude wat het doet. Nuttig als je bestaande scripts begrijpt of code van een collega wil doorgronden.
Klik voor uitleg
Debuggen en verbeteren
Als code niet werkt, kan Claude de fout opsporen en uitleggen waarom. Ook kan hij code efficiënter of leesbaarder maken.
