# Transcriptie: C2 — Desktop-app & Cowork: Claude op je computer

## Wat leer je in deze module?
Je leert wat de Claude desktop-app is, hoe hij verschilt van de web-app, en hoe je via Cowork echte taken laat uitvoeren op je computer. Na deze module kun je de desktop-app zelfstandig gebruiken en weet je hoe je Claude taken laat uitvoeren — niet alleen laten antwoorden.

## Wat is de desktop-app?

De Claude desktop-app is een zelfstandig programma dat je installeert op je Mac of Windows-computer. Hij werkt onafhankelijk van je browser en staat altijd klaar via de taakbalk of menubalk — geen browser openen, geen tabblad zoeken.

Je downloadt hem via claude.ai/download. Installatie duurt minder dan een minuut. Je logt in met hetzelfde account als de web-app. Alle gesprekken en Projects zijn op beide beschikbaar.

## Drie functies: Chat, Cowork en Code

**Chat** is hetzelfde gespreksvenster als de web-app. Typen, vragen stellen, bestanden uploaden. Het voordeel: de app laadt sneller, is altijd beschikbaar en kan bestanden op je computer lezen zonder dat je ze hoeft te uploaden — je geeft eenmalig toegang tot een map en Claude kan de inhoud gebruiken in je gesprekken.

**Cowork** gaat verder. Claude voert taken daadwerkelijk uit op je computer: bestanden lezen, mappen aanmaken, documenten bewerken, bestanden opslaan. Jij geeft een opdracht in gewone taal, Claude maakt een plan, jij keurt goed, Claude voert uit. Altijd met jouw bevestiging.

**Code** is voor het schrijven, uitleggen en debuggen van code. Relevant als je interne tools of automatiseringen wilt bouwen. Als je niet programmeert, gebruik je deze functie niet — maar weet dat hij bestaat.

## Wanneer gebruik je de desktop-app versus de web-app?

Gebruik de desktop-app als je Claude regelmatig gebruikt en hem snel wilt kunnen openen, als je bestanden op je computer wilt laten verwerken zonder ze handmatig te uploaden, of als je Cowork wilt gebruiken voor bestandsbeheer. Gebruik de web-app als je snel iets wilt vragen zonder iets te installeren, op een ander apparaat werkt, of samenwerkt via Teams.

## Cowork: hoe werkt het in de praktijk?

Je opent de desktop-app en kiest voor Cowork. Je geeft Claude toegang tot een map op je computer. Daarna geef je een opdracht in gewone taal.

Voorbeeld voor een makelaarskantoor: "Lees alle PDF-bestanden in de map Bezichtigingen van deze maand en maak voor elk bestand een korte samenvatting als tekstbestand in de map Samenvattingen."

Claude laat je eerst zien wat hij van plan is. Je bevestigt — dan voert hij de stappen één voor één uit. Je ziet wat hij doet.

## Welke taken zijn geschikt voor Cowork?

Cowork werkt goed voor het verwerken van grote aantallen bestanden (samenvatten, hernoemen, sorteren), het aanmaken van mappenstructuren, het omzetten van bestandsformaten en het doorzoeken van mappen op specifieke informatie.

Cowork werkt minder goed voor taken waarbij je specifieke software nodig hebt (Excel-formules berekenen, presentaties opmaken), taken waarbij internettoegang nodig is, en alles wat real-time data vereist.

## Altijd in controle

Claude vraagt altijd om bevestiging voordat hij iets doet dat niet ongedaan gemaakt kan worden — zoals een bestand overschrijven of verwijderen. Je kunt op elk moment stoppen.

Maak altijd een back-up van belangrijke bestanden voordat je Claude op een grote map loslaat. Niet omdat hij fouten maakt, maar omdat automatisering altijd risico's heeft en voorzichtigheid een goede gewoonte is.

Cowork heeft geen toegang tot je e-mail, je CRM of andere systemen tenzij je die expliciet koppelt via MCP — meer daarover in module E1.

## Samenvatting

De desktop-app heeft drie functies: Chat (hetzelfde als de web-app, maar sneller en met directe bestandstoegang), Cowork (Claude voert taken uit op je computer) en Code (voor het schrijven en verbeteren van code). Cowork is geschikt voor bestandsverwerking, mapbeheer en repetitieve documenttaken. Claude vraagt altijd om bevestiging bij ingrijpende acties. Het verschil met gewone Chat: bij Chat geef je een antwoord dat jij dan zelf uitvoert — bij Cowork voert Claude het zelf uit.
