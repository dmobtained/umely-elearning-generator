# Transcriptie: E7 — Portfolio website bouwen met Claude

## Wat leer je in deze module?
Je leert hoe je met Claude Code een eenvoudige website bouwt — zonder kennis van webontwikkeling. Na deze module heb je een werkende portfolio- of bedrijfswebsite die je zelf kunt aanpassen en online kunt zetten.

## Waarom een website bouwen met Claude?

Een eenvoudige website bouwen kostte vroeger duizenden euro's en een developer. Met Claude Code kun je een nette, functionele website maken in een middag — ook als je nog nooit een regel code hebt geschreven. Je beschrijft wat je wilt, Claude schrijft de code, jij controleert het resultaat in je browser.

Dit is niet bedoeld voor complexe webapplicaties of webshops met betalingen. Het is bedoeld voor een eenvoudige aanwezigheid online: wie ben je, wat doe je, hoe kun je contact opnemen.

## Wat heb je nodig?

- Claude Code geïnstalleerd (zie module D1)
- Een tekstverwerker of code-editor (VS Code is gratis en werkt goed)
- Een webbrowser om het resultaat te bekijken
- Voor online zetten: een hostingaccount (bijv. Netlify of GitHub Pages, beide gratis)

## Stap 1: Beschrijf wat je wilt

Open Claude Code en beschrijf je website in gewone taal:

"Maak een eenvoudige portfolio website voor een makelaarskantoor. De website heeft een homepage met een hero-sectie met de tekst 'Uw thuis, onze passie', een sectie met drie diensten (Aankoop, Verkoop, Verhuur), een over-ons sectie en een contactformulier. Gebruik een professionele uitstraling met donkerblauwe en witte kleuren."

Claude genereert de HTML, CSS en eventueel JavaScript. Hij slaat de bestanden op in een map die jij aangeeft.

## Stap 2: Bekijk het resultaat

Open het gegenereerde HTML-bestand in je browser. Je ziet direct hoe de website eruitziet. Is er iets wat je wilt aanpassen? Vertel het Claude:

"Verander de achtergrondkleur van de header naar #1a2a4a. Maak de tekst in de hero-sectie groter. Voeg een telefoonnummer toe aan de contactsectie."

Claude past de code aan. Je vernieuwt de browser en ziet het resultaat.

## Stap 3: Verfijn totdat je tevreden bent

Dit is een iteratief proces. Geef Claude feedback, hij past aan, jij bekijkt, herhaal. Typische aanpassingen:

- Teksten aanpassen
- Kleuren veranderen
- Secties toevoegen of verwijderen
- Afbeeldingen invoegen (Claude geeft aan waar je ze moet plaatsen)
- Mobiele weergave verbeteren

## Stap 4: Online zetten

Als je tevreden bent, zet je de website online. De eenvoudigste gratis optie is Netlify:

1. Maak een gratis account aan op netlify.com
2. Sleep je websitemap naar het Netlify-dashboard
3. Netlify geeft je een URL — je website is live

Voor een eigen domeinnaam (bijv. www.jouwbedrijf.nl) heb je een domeinnaam nodig die je koopt bij een registrar (bijv. TransIP of Antagonist). Dit kost ongeveer €10-15 per jaar.

## Wat kan Claude niet voor je website?

Claude kan geen betalingssystemen integreren, geen inlogsystemen bouwen en geen databases koppelen — dat vereist meer technische setup. Voor een eenvoudige informatieve website is dat niet nodig.

Claude kan ook geen afbeeldingen genereren of uploaden. Je voegt eigen afbeeldingen toe door ze in de juiste map te plaatsen en Claude te vragen ze in te voegen.

## Beperking: onderhoud

Een website die Claude heeft gebouwd, moet je zelf onderhouden. Als je teksten wilt aanpassen, doe je dat via Claude Code of door de HTML-bestanden direct te bewerken. Als je website complex wordt, overweeg dan een CMS (Content Management System) zoals WordPress — maar dat valt buiten het bereik van deze module.

## Samenvatting

Met Claude Code kun je een eenvoudige bedrijfswebsite bouwen zonder technische kennis — door te beschrijven wat je wilt en het resultaat in je browser te bekijken. Het proces is iteratief: beschrijf, bekijk, verfijn. Online zetten kan gratis via Netlify. Claude kan geen betalingen, logins of databases verwerken — voor een informatieve website is dat niet nodig.
