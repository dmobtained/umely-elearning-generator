# Transcriptie: C2 — Desktop-app: Chat, Cowork en Code

## Wat leer je in deze module?
Je leert wat de Claude desktop-app is, hoe hij verschilt van de web-app, en wat de drie hoofdfuncties zijn: Chat, Cowork en Code. Na deze module weet je wanneer je de desktop-app gebruikt en wat je ermee kunt doen.

## Wat is de desktop-app?

De Claude desktop-app is een zelfstandig programma dat je installeert op je Mac of Windows-computer. Hij werkt grotendeels hetzelfde als de web-app maar heeft een paar extra mogelijkheden — waaronder directe toegang tot bestanden op je computer en integratie met andere programma's via MCP (Model Context Protocol, meer daarover in module E1).

Je download de app via claude.ai/download. Na installatie log je in met hetzelfde account als de web-app.

## Chat

Chat is de standaardfunctie — hetzelfde gespreksvenster als in de web-app. Je typt een vraag, Claude antwoordt. Je kunt bestanden uploaden, gesprekken organiseren via Projects en gebruik maken van geheugen.

Het voordeel van de desktop-app boven de web-app: de app laadt sneller, is altijd beschikbaar via je taakbalk en kan bestanden op je computer lezen zonder dat je ze eerst moet uploaden (als je dat inschakelt).

## Cowork

Cowork is een functie waarbij Claude actief taken voor je uitvoert op je computer — bestanden lezen, mappen aanmaken, documenten bewerken. Dit gaat verder dan alleen antwoorden geven: Claude neemt stappen op je computer.

Dit is krachtig maar vereist aandacht. Claude vraagt altijd om bevestiging voordat hij iets uitvoert wat niet ongedaan gemaakt kan worden. Lees altijd wat hij van plan is voordat je bevestigt.

Praktisch voorbeeld: je vraagt Claude om alle PDF-rapporten in een map samen te vatten en de samenvattingen op te slaan als tekstbestanden. Claude doet dat stap voor stap, met jouw goedkeuring per stap.

Beperking: Cowork werkt het beste voor bestandsbeheer en documentverwerking. Het is geen volledige computerbesturing — Claude kan geen programma's openen, geen e-mails versturen en geen websites bezoeken tenzij je hem daar expliciete toegang voor geeft.

## Code

De Code-functie in de desktop-app is specifiek voor mensen die code schrijven of willen leren schrijven. Claude kan code genereren, uitleggen, debuggen en verbeteren. Dit is relevant voor bedrijven die intern tools willen bouwen of automatiseringen willen opzetten.

Voor niet-developers: je hoeft hier niets mee te doen. Als je niet programmeert, gebruik je deze functie niet. Maar weet dat hij bestaat — als je op een punt komt dat je wilt automatiseren, is dit waar je begint.

## Wanneer gebruik je de desktop-app versus de web-app?

Gebruik de desktop-app als je:
- Claude regelmatig gebruikt en hem snel wilt kunnen openen
- Bestanden op je computer wilt laten verwerken zonder ze handmatig te uploaden
- Cowork wilt gebruiken voor bestandsbeheer

Gebruik de web-app als je:
- Snel iets wilt vragen zonder iets te installeren
- Op een ander apparaat werkt
- Samenwerkt met collega's via Teams

## Samenvatting

De desktop-app heeft drie functies: Chat (hetzelfde als de web-app), Cowork (Claude voert taken uit op je computer) en Code (voor het schrijven en verbeteren van code). De desktop-app is handig als je Claude regelmatig gebruikt en meer wilt dan alleen een gespreksvenster. Cowork is krachtig maar vereist dat je altijd meekijkt met wat Claude van plan is.
